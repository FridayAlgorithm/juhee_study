# ✏️알고리즘 스터디

## ✨ 스터디 방식

* 매일 2문제씩 풀기 (주말은 토,일 포함 2문제)
* 고정 리뷰 시간 : 금요일 오후 7시 / 공휴일 리뷰 시간 : 금요일 오전 11시
* 구글 meet로 진행
* 리뷰는 당일 사다리타기
* 코드업 기초 100(개인별) + 유형별 2문제 이상
* 코드업 이후 프로그래머스 LV.2~3 위주로 풀이
